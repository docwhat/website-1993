Christian Höltje's Magic Tools

Welcome to my Magic Tools. Most of these tools are/will be written in perl, since perl is fairly portable across platforms.

True, this collection is a little sparse right now, but I'll keep writing more. Plus, If you ask for something, I'll probably write it!

Magic HTML Tools

First, I want mention WebLint, a really nice Web Validator. I use it alot.

Here are the tools as they stand:

    FixHTML.pl

    This script is used to rename all files in your ~/public_html directory from *.htm to *.html. Since I edit my html files on a DOS machine, I need to change the extensions. It also makes all the filenames lowercase. I can't test files with upper and lower case names under DOS, so I just make all my files lower case.

    mktoc.pl

    The script is used to generate a Table of Contents for your web pages. Currently, it generates both an HTML and a Graphics Table of Contents. It should work equally well under both DOS and UNIX.
